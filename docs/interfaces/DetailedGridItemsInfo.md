# Interface: DetailedGridItemsInfo

Information about a grid item's placement.

Specifies which grid lines the item spans on both axes.
Line numbers are 1-indexed, with 1 being the first line.

## Properties

| Property                                 | Type     | Description                             |
| ---------------------------------------- | -------- | --------------------------------------- |
| <a id="column_end"></a> `column_end`     | `number` | Ending column line number (exclusive)   |
| <a id="column_start"></a> `column_start` | `number` | Starting column line number (1-indexed) |
| <a id="row_end"></a> `row_end`           | `number` | Ending row line number (exclusive)      |
| <a id="row_start"></a> `row_start`       | `number` | Starting row line number (1-indexed)    |
