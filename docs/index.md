# Taffy Layout

Welcome to the official documentation for Taffy Layout.

Taffy is a layout engine that supports both Flexbox and CSS Grid. It is designed to be high-performance, lightweight, and easy to use across different platforms.

## Key Goals

- **Correctness**: Adhere strictly to the CSS specifications.
- **Performance**: Optimized for speed and low memory usage.
- **Portability**: Easy to integrate into any environment via Rust or WASM.
