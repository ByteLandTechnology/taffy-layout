# GridTemplateArea

```ts
type GridTemplateArea = {
  columnEnd: number;
  columnStart: number;
  name: string;
  rowEnd: number;
  rowStart: number;
};
```

Named grid area definition.

Defines a named area within the grid and its boundaries.

## Properties

| Property                               | Type     | Description               |
| -------------------------------------- | -------- | ------------------------- |
| <a id="columnend"></a> `columnEnd`     | `number` | End column line           |
| <a id="columnstart"></a> `columnStart` | `number` | Start column line         |
| <a id="name"></a> `name`               | `string` | The name of the grid area |
| <a id="rowend"></a> `rowEnd`           | `number` | End row line              |
| <a id="rowstart"></a> `rowStart`       | `number` | Start row line            |
