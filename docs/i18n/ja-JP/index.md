# Taffy Layout

Taffy Layout の公式ドキュメントへようこそ。

Taffy は、Flexbox と CSS Grid の両方をサポートするレイアウトエンジンです。高性能で軽量、かつさまざまなプラットフォームで簡単に使用できるように設計されています。

## 主な目標

- **正確性**: CSS 仕様に厳密に準拠します。
- **パフォーマンス**: 速度と低いメモリ使用量のために最適化されています。
- **移植性**: Rust または WASM を介してあらゆる環境に簡単に統合できます。
