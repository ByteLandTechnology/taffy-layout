# はじめに

Taffy Layout へようこそ！このガイドでは、高性能なレイアウトエンジンの使い方を説明します。

## クイックリンク

- [インストール](./installation.md) - 環境に Taffy をセットアップします。
- [クイックスタート](./quick-start.md) - 数分で独自のレイアウトを作成します。
- [ノードツリーの構築](./building-trees.md) - レイアウトツリーの構築方法を学びます。
- [レイアウトの計算](./computing-layouts.md) - レイアウト計算プロセスをマスターします。
- [設定](./configuration.md) - ニーズに合わせて Taffy を微調整します。

## なぜ Taffy なのか？

Taffy は、クロスプラットフォーム開発者にとって最も高速で信頼性の高いレイアウトエンジンとなるよう設計されています。ゲーム UI、独自のモバイルフレームワーク、または高性能な Web コンポーネントを構築している場合でも、Taffy は必要な標準準拠の Flexbox および Grid ツールを提供します。
