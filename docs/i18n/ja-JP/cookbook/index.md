# レイアウトクックブック

**すぐにコピーできるレイアウトレシピ。**

図を確認し、コードをコピーして、ニーズに合わせて調整してください。

## レシピ

| レシピ                                              | 説明                                                 | 技術スタック |
| :-------------------------------------------------- | :--------------------------------------------------- | :----------- |
| **[フレックス行カード](./flex-row-cards.md)**       | 均一サイズのカードの行。                             | Flexbox      |
| **[グリッドダッシュボード](./grid-dashboard.md)**   | ヘッダー、サイドバー、メインコンテンツエリア。       | Grid         |
| **[Holy Grail レイアウト](./holy-grail-layout.md)** | ヘッダー/フッター付きのクラシックな 3 列レイアウト。 | Flexbox      |
| **[スティッキーフッター](./sticky-footer.md)**      | コンテンツが少なくてもフッターを下部に維持。         | Flexbox      |
| **[メディアカード](./media-card.md)**               | 左側に画像、右側にテキスト。                         | Flexbox      |
