# 入门指南

欢迎使用 Taffy Layout！本指南将帮助您快速上手这款高性能布局引擎。

## 快速链接

- [安装](./installation.md) - 在您的环境中设置 Taffy。
- [快速开始](./quick-start.md) - 在几分钟内创建您的第一个布局。
- [构建节点树](./building-trees.md) - 学习如何构建布局树。
- [计算布局](./computing-layouts.md) - 掌握布局计算过程。
- [配置](./configuration.md) - 根据您的需求微调 Taffy。

## 为什么选择 Taffy？

Taffy 旨在成为跨平台开发者可用的最快、最可靠的布局引擎。无论您是在构建游戏 UI、自定义移动框架还是高性能 Web 组件，Taffy 都能提供您所需的符合标准的 Flexbox 和 Grid 工具。
